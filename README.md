# 🎬 mootti

> 선택의 부담은 줄이고,  
> 취향에 맞는 영화를 빠르게 발견하는 영화 추천 서비스

---

## 📌 프로젝트 소개

1인 가구 사용자가 식사 시간이나 휴식 시간에  
*“무엇을 볼지 고민하다 시간이 지나가는 문제”*를 해결하기 위해 기획된  
**상황·취향·경험 중심 영화 추천 서비스**입니다.

사용자의 선택 피로를 줄이기 위해  
추천 방식을 하나로 고정하지 않고, **상황에 맞는 세 가지 추천 방식**을 제공합니다.

---

## 🎯 핵심 문제 정의 (Persona)

- 1인 가구 사용자는 영상 콘텐츠를 자주 소비하지만  
  선택 과정에서 피로를 느낌
- 선택 시간이 길어질수록  
  몰입도와 만족도가 감소
- *“고민 없이 바로 고를 수 있는 추천 경험”*이 필요

---

## 🧩 주요 기능

### 🔐 User Profile & Community
- 회원가입 및 사용자 정보 관리
- 성별, 나이, 선호 장르 입력
- 좋아요 / 볼 영화 / 본 영화 리스트 관리
- 영화 리뷰 및 댓글 작성
- 리뷰 좋아요 및 리뷰 수 기반 티어 시스템

---

### 🤖 Recommendation Systems

#### 1️⃣ Blind Review Recommendation
- 영화 정보 없이 리뷰만 보고 선택
- 단 하나의 영화 선택 후 상세 정보 공개
- 선택 피로 최소화 및 몰입도 강화

#### 2️⃣ Keyword-Based Recommendation
- 사용자의 기분·상황을 자연어로 입력
- OpenAI API 기반 상황 맞춤 영화 추천
- 취향 데이터가 없어도 즉시 사용 가능

#### 3️⃣ Movie Worldcup Recommendation
- 영화 선택 과정을 게임화
- 랜덤 월드컵 / 사용자 선호 기반 월드컵 제공
- 선택 과정을 통해 취향 데이터 자연스럽게 수집

---

## 🔄 데이터 선순환 구조

- 사용자 선택 · 좋아요 · 리뷰 · 월드컵 결과  
→ 취향 데이터 축적  
→ 추천 정확도 향상  
→ 더 나은 사용자 경험 제공

리뷰 티어 시스템을 통해  
커뮤니티 참여와 신뢰도를 함께 강화합니다.

---

## 🤖 AI 활용 전략

- OpenAI API 활용
  - 취향·상황 기반 영화 후보 생성
  - 키워드 기반 추천
  - 영화 월드컵 후보 구성
- 서비스 초기 단계에서는  
  **AI-Generated Review Bootstrapping**을 통해  
  리뷰 데이터 콜드스타트 문제 해결

> AI는 선택을 대신하는 것이 아니라,  
> 사용자의 선택을 돕는 역할에 집중합니다.
